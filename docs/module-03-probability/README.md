# Module 03：概率论基础

## 模块简介
建立随机变量与分布的基本概念，理解概率计算和常见分布，为统计推断与机器学习模型概率视角奠基。

## 学习目标
- 能解释随机试验、样本空间、事件与概率公理。
- 能熟悉常见离散/连续分布（Bernoulli、Binomial、Poisson、Normal）。
- 能计算期望、方差、协方差与相关系数，理解其意义。
- 能应用条件概率、全概率公式与 Bayes 定理解决简单问题。
- 能用模拟方式验证大数定律，绘制分布曲线。

## 核心概念
- 随机变量与分布：PMF/PDF/CDF 描述结果的概率结构。
- 期望与方差：衡量集中趋势与离散程度。
- 协方差与相关：度量变量间线性关系。
- 条件概率与 Bayes：更新信念、反向推断。

## 推荐学习顺序
1. 概率公理与事件运算。
2. 离散分布（Bernoulli、Binomial、Poisson）。
3. 连续分布（Uniform、Normal、Exponential），理解 PDF/CDF。
4. 期望、方差、协方差与相关系数。
5. 条件概率、全概率公式与 Bayes 定理。
6. 大数定律与中心极限定理的直观与模拟。

## 配套编程练习
- 用 numpy 模拟抛硬币、掷骰子，统计频率并与理论概率对比。
- 绘制不同分布的 PMF/PDF/CDF，观察参数变化的影响。
- 模拟大数定律：增加样本数量，观察样本均值收敛。
- 简单 Bayes 例子：给定先验与似然，计算后验分布。

## 与机器学习的联系
- 朴素 Bayes、概率图模型等直接基于概率论。
- 概率视角有助于理解模型的不确定性与正则化。
- 分布假设贯穿模型评估与数据生成。
