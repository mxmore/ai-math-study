# Module 01：函数与微积分基础

## 模块简介
回顾函数、极限、导数与多元微积分，为理解梯度、链式法则和优化直觉打下基础。

## 学习目标
- 能解释函数、极限与连续性的含义，并能判断简单函数的极限。
- 能计算一元函数导数并理解几何意义，掌握链式法则。
- 能求多元函数的偏导数与梯度，并理解方向导数。
- 能通过泰勒展开理解近似思想，知道误差阶的意义。
- 能用 Python 数值方法近似导数并与解析解对比。

## 核心概念
- 极限与连续：描述函数在某点附近的行为。
- 导数与几何意义：切线斜率、变化率；链式法则处理复合函数。
- 多元函数与梯度：梯度指向最快上升方向，方向导数衡量任意方向变化率。
- 泰勒展开：用多项式局部近似函数，理解一阶/二阶项对误差的影响。

## 推荐学习顺序
1. 回顾函数、极限与连续性概念。
2. 一元微分：基本求导公式、链式法则。
3. 多元微分：偏导、梯度、方向导数。
4. 泰勒展开与近似直觉。
5. 数值求导与误差分析。

## 配套编程练习
- 在 Notebook 中实现数值差分求导，验证 `sin(x)`、`exp(x)` 的导数。
- 可视化一元函数及其切线斜率随点的变化。
- 对二元函数绘制等高线并叠加梯度方向箭头。
- 使用泰勒展开近似 `exp(x)`，比较不同阶数的误差。

## 与机器学习的联系
- 梯度是优化算法（如梯度下降）的核心。
- 多元函数的偏导与链式法则支撑反向传播思想。
- 近似思想有助于理解优化中的局部线性化。
