# Module 04：数理统计与估计理论

## 模块简介
介绍抽样、估计与假设检验的核心思想，理解如何从样本推断总体，为模型评估与 A/B 测试提供基础。

## 学习目标
- 能区分总体与样本，理解抽样分布的概念。
- 能计算样本均值、方差并理解其统计意义。
- 能构建常见分布参数的极大似然估计（MLE）与 MAP 估计。
- 能理解置信区间与假设检验的基本流程与含义。
- 能用模拟数据完成 A/B 测试的简单假设检验。

## 核心概念
- 抽样与抽样分布：统计量在重复抽样下的分布行为。
- 点估计：MLE、MAP、偏差与方差。
- 区间估计：置信区间的含义与构造思路。
- 假设检验：原假设、备择假设、显著性水平、p 值、第一类与第二类错误。

## 推荐学习顺序
1. 总体、样本与抽样分布；中心极限定理的意义。
2. 点估计方法：矩估计、MLE、MAP（高层直觉）。
3. 区间估计：均值的置信区间示例。
4. 假设检验流程：以 A/B 测试为例，理解 p 值与显著性水平。
5. 实践：用模拟数据实现估计与检验。

## 配套编程练习
- 使用 numpy 生成正态样本，估计均值与方差，对比真实参数。
- 通过数值方式求解正态分布参数的 MLE。
- 构造简单的 95% 置信区间，并用重复实验观察覆盖率。
- 模拟 A/B 测试点击率数据，完成假设检验并解释结果。

## 与机器学习的联系
- 模型参数估计通常基于似然或后验最大化。
- 置信区间与假设检验思想有助于模型评估与对比实验。
- MAP 与正则化的联系：先验相当于正则项。
